Why ws?

🌿 Branch-Scoped

Each git branch has its own working set. Switch branches, your context switches too. No manual cleanup.

🤖 AI-Native

Built for AI pair programming with Claude Code, Cursor, and Copilot. Give AI focused context.

💾 Persistent

Working sets survive terminal sessions. Open ws tomorrow — your files are still there.

📂 Git-Aware

Shows inline git status. Auto-adds modified files. Knows your repo better than you do.

🔍 Fuzzy Search

Press / to instantly filter your working set. Find files in milliseconds.

⚡ Fast

Built with Go + Bubbletea. Startup in <100ms. Zero dependencies. Single binary.

See ws in Action

Installation

🍺 Homebrew

macOS & Linux

brew tap n-filatov/tap && brew install ws

📦 APT

Debian & Ubuntu

curl -fsSL https://n-filatov.github.io/ws/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/ws.gpg && echo "deb [signed-by=/usr/share/keyrings/ws.gpg] https://n-filatov.github.io/ws ./" | sudo tee /etc/apt/sources.list.d/ws.list && sudo apt update && sudo apt install ws

🔷 Go

Any platform

go install github.com/n-filatov/ws@latest

📜 Source

Build from source

git clone https://github.com/n-filatov/ws && cd ws && make install
